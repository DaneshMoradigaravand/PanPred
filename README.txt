`Program: PanPred (Machine Learning tools to predict antibiotic resistance from NGS data)
Version: 0.1
Contact: Danesh Moradigaravand <d.morady@gmail.com>

Usage: PanPred <command> [options]

Command:
                preprocess	Preprocess and creates the NGS input data for Machine Learning tools
		predict		Calls a Machine Learning model, i.e. Logistic regression, RandomForests, GradientBossting and Deep Learning, to predict resistance from input
		interpret	Interpret and plot the output of the prediction stage
